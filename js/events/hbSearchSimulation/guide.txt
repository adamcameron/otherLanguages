page load
	-> load settings
	-> get properties
		-> do CHANGEPAGE(1)
			-> activate controls (stops listening for change page)
                                                                
CHANGEPAGE
	-> change page setting (page=n)
		-> render page
		-> render pagination
                                                                                

change sort
	-> change setting (sort)
		-> sort
			-> do CHANGEPAGE(1)
                                                     
change filter
	-> change settings (filters)
		-> filter
			-> sort


resize page
	-> change page size setting
		-> do CHANGEPAGE(1)
